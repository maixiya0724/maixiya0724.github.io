js class 类

2019年12月10日 338 字 大概 1 分钟 技术 技术

js class 是es新增的语法. 文档

class 的基本语法

1.es6的class 就是es5构造函数的另一种写法.因为es的构造函数写法跟其他的面向对象语言差异很大，很容易让新学这门语言的程序猿干到困惑，所以es6提供了更接近传统语言的写法，引入了class 类这个概念，座位对象的模板，通过class关键子可以定义类。例如
class Point {
  constructor(x, y) {
    this.x = x;
    this.y = y;
  }

  toString() {
    return '(' + this.x + ', ' + this.y + ')';
  }
}

constructor其实就是构造方法，而this关键字代表实例对象，也就是说，es5的构造函数Point对应es6的Point类的构造方法.

2.构造函数的proitotype属性，在es6上面同样存在，事实上，类的所有方法都定义在类的prototype上.
class Point {
  constructor() {
    // ...
  }

  toString() {
    // ...
  }

  toValue() {
    // ...
  }
}

// 等同于

Point.prototype = {
  constructor() {},
  toString() {},
  toValue() {},
};

3.constructor 方法

constructor方法是类的默认方法，通过new命令生成对象实例时，自动调用该方法。一个类必须有constructor方法，如果没有显式定义，一个空的constructor方法会被默认添加。

4.实现继承

Class可以通过extends关键字实现继承，这比es5的通过修改原型链实现集成，要清晰和方便很多….
class Point {
}

class ColorPoint extends Point {

}
