eslit 配合vscode 自动修复语法错误

秋风清，秋月明，落叶聚还散，寒鸦栖复惊，相知相见知何日，此时此刻难为情 – 金庸

eslint 是对语法规则进行检查和约束的一种规范，这个更利于代码的可读性和规范性

安装

npm install eslint --save-dev

配置文件 : .eslintrc.js 在配置文件里面配置具体的eslint规则

vscode 配合eslint保存自动格式化

vscode 需要安装插件 

ESLint vscode 代码提示错误
vuetur  Vue 开发插件 格式化 代码段
prettier - code formatter 代码格式化插件，非常牛，可以自定义格式化的配置

vscode setting.json 配置 

{
   ...
   "vetur.format.defaultFormatter.js": "vscode-typescript",
   "vetur.format.defaultFormatter.html": "js-beautify-html",
   "eslint.validate": [
   "javascript",
   "javascriptreact",
   {
   "language": "html",
   "autoFix": true
   },
   {
   "language": "vue",
   "autoFix": true
   }
   ],
   // 保存自动修复
   "eslint.autoFixOnSave": true,
   // jsx自动修复有问题，取消js的format
   "editor.formatOnSave": false,
   // Enable/disable default JavaScript formatter (For Prettier)
   "javascript.format.enable": false,
   "prettier.singleQuote": true,
   // 点击保存时，根据 eslint 规则自定修复，同时集成 prettier 到 eslint 中
   "prettier.eslintIntegration": true,
   ...
}
