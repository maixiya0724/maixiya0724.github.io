vue指令

2020年06月22日 300 字 大概 1 分钟 技术 技术

可惜没如果

vue指令 官方文档

除了核心功能默认内置的指令 (v-model 和 v-show)，Vue 也允许注册自定义指令。注意，在 Vue2.0 中，代码复用和抽象的主要形式是组件。然而，有的情况下，你仍然需要对普通 DOM 元素进行底层操作，这时候就会用到自定义指令。当需要对普通DOM进行底层操作时，可能会用到自定义指令;

注册全局指令

指令有两种注册方式一种是全局注册 ，另一种是在组件内局部注册,全局指令全局都可以使用，局部指令只能在当前的组件上使用.

全局注册

// 注册一个全局自定义指令 `v-focus`
Vue.directive('focus', {
  // 当被绑定的元素插入到 DOM 中时……
  inserted: function (el) {
    // 聚焦元素
    el.focus()
  }
})

组件内注册

//如果想注册局部指令，组件中也接受一个 directives 的选项：
directives: {
  focus: {
    // 指令的定义
    inserted: function (el) {
      el.focus()
    }
  }
}

使用

你可以在模板的任何元素上使用新的v-focus property
<input v-focus>
